📚 What is AmpliPi?

AmpliPi is an open-source, multi-room/zone home audio controller and amplifier designed for the modern smart home.

🏗️ Architecture

Built around a Raspberry Pi Compute Module 3+, AmpliPi integrates high-quality DACs, ADCs, and Class-D amplifiers into a single 1U rack-mountable unit. It supports 4 audio sources and 6 stereo zones (expandable to 36 zones).

🎵 Audio Sources

Stream from Spotify, Pandora, AirPlay, DLNA, Internet Radio, or connect analog sources via RCA inputs. Each zone can play different sources simultaneously with independent volume control.

📱 Control Options

Control via responsive web app, native iOS/Android apps, REST API, or integrate with Home Assistant, OpenHAB, and other automation platforms. Access locally at amplipi.local when on the same network.

⚡ Power & Quality

Each zone delivers 60W of clean power using TPA3255 Class-D amplifiers. High-quality PCM5102A DACs ensure pristine audio reproduction with SNR >100dB.

🔧 Open Source

Fully open hardware and software design. Customize, modify, and contribute to the project. Built on Linux with Python-based control software.

🏠 Smart Integration

RESTful API enables seamless integration with smart home systems. Create scenes, automations, and voice control through Alexa, Google Assistant, or Siri Shortcuts.
